## 文字列 S のすべての文字が数字（0〜9）で構成されているかを確認し、数字のみなら “YES”、それ以外の文字が含まれていれば “NO” を出力

```ruby
s = gets.chomp

if s.match?(/\A\d+\z/)
  puts "YES"
else
  puts "NO"
end
```
•	S.match?(/\A\d+\z/) の正規表現で、S が数字のみで構成されているかチェックできます。
•	\A は文字列の先頭、\z は文字列の末尾を示し、^\d+$ に似た形で、全体が数字だけで構成されていることを表す。
•	\d+ は、1文字以上の数字（0〜9）が続くことを意味する。
